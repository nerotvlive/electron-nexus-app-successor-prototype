# Zyneon Desktop Electron App (discontinued)

[Zyneon Desktop](https://github.com/zyneonstudios/desktop) by **Zyneon Apex** is the successor of the [NEXUS App/Zyneon App](https://github.com/nerotvlive/nexus-app) by **Zyneon Nexus**.


## What is this repository?

After the decision was made to replace the NEXUS app, prototypes were created using several stacks. This Electron, Bootstrap, and HTML/CSS/JS stack was one of them. Zyneon Desktop was not implemented using this stack.


## But what about Zyneon Desktop? (continued)

Zyneon Desktop was implemented using technologies such as Tauri, Rust, Vite, Vue, Tailwind (and more...) and can be found here: https://github.com/zyneonstudios/desktop
