# Zyneon Desktop

This repository contains the desktop client for Zyneon (mods).<br>
A new-gen modding utility and platform. Successor of the NEXUS App<br>
**Warning:** This is a very early alpha version and is not ready for production use. There are a lot of bugs and missing features.
<br>
**Note:** This project is still in active development and may undergo significant changes. Use at your own risk.
